Responsive Web Design

Let's start with Responsive Web Design.

I'm quite sure, you run into this picture in some form before. It's the typical marketing image to depict a web app, that is usable on different devices by adjusting the layout accordingly.

But what if I told you that this approach could fell short?

How would you classify this Samsung Galaxy Note 3 with a pen? Is it a mobile? It is small enough for one. But you have a pointer device; the pen. So designing it for desktop? You would waste real-estate on the screen after all.

While I was preparing for the GameDev.js jam in early 2020 I ran across a MDN article regarding sensor APIs. I was aware of the ability to recognise motion and orientation from my days with Firefox OS. But by now, there are more. This talk will show you some tech demos. They're not ready for production usage in general, since you need to toggle a flag in your browser to get access to the underlying API. But that shouldn't stop you to show off, what's possible on the web :-)

Detectar

Detectar

After the talk I decided to create a small demo site, which runs some primitive checks on your device capabilities.

Since then, I started to design tooltips based on a hover media query instead of guessing from the device width. I'm not quite sure what the colours media query does. I can think of use cases for optimising for eBook readers, though. Orientation can be used if you have certain criteria. For example, showing a sidebar. Device Motion and Orientation have we seen just now. They got replaced by a Permission-based API. Accelerometer measures the velocity. Ambient Light the light level around the devices. I have a small demo for you here, too. Web Bluetooth is available on Chrome behind a flag. I've also got something prepared here. Geolocation should be known from map apps. Gyroscope tells you something about the position in space of the device. We saw it in the user activity demo. For example, when you have indication that the user is driving, perhaps offer to read alout your blog post? Magnetometer is measuring the magnet field of the earth. I think, you should be able to measure the altitude roughly? NFC or Near-Field Communication can be used on slow distances. I have nothing similar available.

Project Fugu

Platform Adjacency Theory

Fugu status page

Let's talk about Project Fugu for a moment.

It's an initiative at the Chromium project to push the web forward by providing access to more sensor data. For one, to catch up with native apps. But also to enable new scenarios, we cannot even think of! There's some pushback from Apple and Mozilla here. If you are interested, read this blog post by Alex Russell.
